# Task 8 - Simple Blog Layout Using Bootstrap 5

This project is a responsive blog layout built using Bootstrap 5 CDN.

## Features
- Responsive Navbar
- Blog Cards with images
- Bootstrap Grid System
- Footer with social icons
- Utility classes for spacing and styling

## Concepts Covered
- Bootstrap 5 CDN
- Grid System
- Bootstrap Components (Navbar, Card)
- Utility Classes
- Responsive Design

## How to Run
Open index.html in browser.
